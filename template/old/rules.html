Rules

Users

  1. There are two different types of users in FLEQ: administrators and players.
  2. By default, a new user registered will be a player user (more info).
  3. An administrator is a user who can administer tournaments, although he/she could participate in other as player in other tournaments too.
  4. What kind of actions can a player do?
    • Join a tournament.
    • Obtain details about a tournament.
    • Obtain details about a game.
    • Modify hour and date with his preferences for next games.
    • Play a game.
    • Request a question to be reviewed by the administrator.
    • Check the status of a question review.
  5. What kind of actions can an administrator do? All the ones that a player can do and...
    • Upload questions.
    • Create new tournaments.
    • Resolve a question review.
    • See information about rounds, games, scores about the tournaments he/she administers.

Tournaments

  1. A tournament can be created only by one administrator.
  2. Each tournament is composed by one or more categories of questions.
  3. Each tournament has a start date given by the administrator; the end date depends of number of rounds (and the days for each round).
  4. A player can join a tournament until the start date.
  5. If at the start date of a tournament only one player has registered for it, the tournament is cancelled and the administrator and player are notified.

Rounds

  1. Each round has as duration the number days selected by the administrator when configuring the tournament.
  2. At the beginning of the round, FLEQ defines the games between players.
  3. To equalize the level of opponents, games between players are chosen using the swiss-system.
  4. If the number of players is odd, in each round a player does not play and stays in a round of reflection. The system will add 1 point to his tournament score; in the next round the player will play again.

Games

  1. By default, the game starting hour and date is configured by the administrator.
  2. Players can select other dates and times for a match in his preferences. If the preferences of both players match, FLEQ changes date and time of the game.
  3. The first player to answer 10 questions correctly is the winner of a quizbowl game.
  4. The winner of a game adds 1 point to his tournament score.
